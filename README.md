# User Experience (UX)

<a name="ux"></a>

- ## Target Audience
  
  Our website appeals to people who love photography, appreciate art, and those looking for professional photography services.

- ## Project Goals
  1. Showcase Majka Em's photography portfolio to highlight her skills and expertise in capturing memorable moments.
  2. Create an engaging platform for visitors to explore, interact with, and appreciate stunning visual content.
  3. Provide a seamless user experience that encourages visitors to inquire about photography services and engage with the community.

- ## Users stories

    - ###  First Time Visitor Goals
 
  1. Read testimonials to learn about others' experiences.
  2. Visit the portfolio to explore the photographer's work.
  3. Learn more about the photographer.

    - ### Returning Visitor Goals
 
  1. Discover new pictures added to the portfolio.
  2. Engage by filling out the sign-up page for updates or newsletters.

    - ### Frequent User Goals
 
  1. Stay up-to-date with new photos added to the portfolio.
  2. Provide feedback and suggestions for improvements.

  4. Integration of an online booking system for scheduling photography sessions and consultations directly through the website.

# Deployment

<a name="deployment"></a>

- This website was deployed to GitHub, steps are as follow:
  1. Go to the Settings tab in your GitHub repository.
  2. On the left side of the screen, click on Pages.
  3. In the source section drop-down menu, choose the Master Branch.
  4. After selecting the master branch, the page will automatically refresh, showing a detailed ribbon display to confirm the successful deployment.
